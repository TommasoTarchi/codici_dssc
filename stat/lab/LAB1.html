Working with random variables

Gamma distribution

Let \(X\) distributed according to a gamma distribution with shape parameter \(\alpha\) and rate parameter \(\theta\). The probability density function (pdf) of \(X\sim Gamma(\alpha, \theta)\) is \[f_X(x)= \frac{\theta^{\alpha}}{\Gamma(\alpha)} x^{\alpha-1}e^{-\theta x}, \quad \alpha, \, \theta > 0, \quad x>0 \] There exists an equivalent parametrisation in terms of shape and scale parameter (\(\sigma = 1/\theta\)), so \[ f_X(x)= \frac{1}{\sigma^{\alpha}\Gamma(\alpha)} x^{\alpha-1}e^{- \frac{x}{\sigma}}, \quad \alpha, \, \sigma > 0, \quad x>0\] In R we can work with the gamma distribution by using r,d,p,q followed by gamma, depending on whether we are interested in random generation, pdf, cumulative distribution function (cdf) or quantile. A look on the R help is helpful to analyse the functions that you are using (you can see how to call a function, the arguments need to be passed, the details and the returning values of a function).

Negative Binomial

In the theory lecture you take into account the case where \(X \sim \mathcal{NB}(p,r)\) models the number of trials needed to observe the \(r\)-th success (considering a Bernoulli experiment).

Here we consider the negative binomial used to model the number of failures before a prefixed number of successes (\(r\)) where the probability of success in each Bernoulli trial is \(p\). ( considering a Bernoulli experiment)

The probability mass function of the negative binomial distribution (also Pascal distribution), \(X \sim \mathcal{NB}(p,r)\), is stated as

\[ P(X=x) = \binom{r+x-1}{r-1}p^r(1-p)^k, \quad \quad p \in (0,1), \quad r =1,2,\ldots , \quad x =0,1,2, \ldots \] You could obtain the alternative formulation (see the slides) by taking \(Y=X+r\), that is \(Y \sim \mathcal{NB}(p,r)\), has pmf \[P(Y = y) = \binom{y-1}{r-1}p^r(1-p)^{y-r}, \quad \quad p \in (0,1), \quad r =1,2, \ldots, \quad y = r,r+1, r+2, \ldots \]

Monte Carlo simulation

Simulation is a well-known technique designed to approximate a process and to retrieve general results by assuming to observe the process several times. We rely on the so called Monte Carlo simulation, a wide class of simulation procedures based on sampling independent and identically distributed values from a process precisely, from the underlying presumibly true probability distribution of the process and computing numerical outputs. The steps of a Monte Carlo simulation are:

  • generate \(n\) independent and identically distributed values from a process;

  • compute a summary for this sample, a statistic;

  • repeat the steps above \(R\) times and obtain a sample distribution for the statistic.

Some relationships among probability distributions

Let \(U_i\sim U(0,1)\), \(i=1,\ldots,n\) be independent and identically distributed random variables (from a uniform continuous distribution).

We can verify that the random variable \(X\) defined as: \[X=-\frac{\log(U_1)}{\theta}\] follows an exponential distribution with parameter \(\theta\). (Fix \(\theta = 2\))

R <- 1000
theta<- 2

x <- rep(0,R)
for(i in 1 : R) {
  u <- runif(1, 0, 1)
  x[i] <- -log(u)/theta
}

hist(x, prob = TRUE, breaks = 30)
curve(dexp(x, theta), from = 1e-16, to = 4, add = T, col = "red")

Now we verify that the random variable \(Z=min\{X_1,X_2, \ldots, X_n\}\), where \(X_i=-\log(U_i)/\theta\), follows an exponential distribution with parameter \(n\theta\).

z <- rep(0, R)
n <- 10
for (i in 1:R) {
  u <- runif(n, 0, 1)
  z[i] <- min(-log(u)/theta)
}

hist(z, prob = TRUE, breaks = 30)
curve(dexp(x, n * theta), add = T, lwd = 2, col = "red")

Normal and chi-square distribution

Let \(x_1, x_2, \ldots, x_n\) a random sample simulated from \(X\sim N(0,1)\) and \(T_n=\sum_{i=1}^{n}X_i^2\). Using a Monte Carlo simulation study, verify that \(T_n\) follows a chi-square distribution with n degrees of freedom. (Consider \(n=10\))

n <- 10
R <- 10000
chisq.mc <- rep(0, R)
for (i in 1 : R){
  x <- rnorm(n = n, mean = 0, sd = 1)
  chisq.mc[i] <- sum(x^2)
 }

hist(chisq.mc, probability = TRUE, xlab = "x", breaks = 30)
curve(dchisq(x = x, df = n), add = TRUE, col = 2)

Compare the empirical cumulative distribution function with the cumulative distribution function.

chisq.ECDF <- ecdf(chisq.mc)
curve(chisq.ECDF(x), from = 0, to = 25, xlab = "x")
curve(pchisq(q = x, df = n), add = TRUE, col = 2)
